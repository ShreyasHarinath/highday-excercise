# HIGHDAY-Exercise

## Project setup
```
yarn install
```
_or_
```
npm install
```

### Compiles and hot-reloads for development
```
yarn serve
```
_or_
```
npm run serve
```

### Compiles and minifies for production
```
yarn build
```
_or_
```
npm run build
```

### Lints and fixes files
```
yarn lint
```

### JSON-Server
```
json-server data.json
```

### Axios
```
yarn add axios
```
_or_
```
npn i axios
```
